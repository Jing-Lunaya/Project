## 关于我
曾经的我,有许多的不太好的情绪,面对事情容易太感感同身受的过于的设身处地为别人着想.
每个人都为了别人而活,我也是那样的一个人.结果没有人真的活着,用这样的方式绑架者彼此.

我在这样的环境里成长成熟. 我站了起来. 成为了自己
不再为了别人而活.只是学着活出我自己的人生.

不再为了第一为了人们眼中的最好最优秀去追求.只是今天做到最好.
做好每个今天.

我依然喜欢使用头脑. 使用头脑在它应该被使用的地方.
在放下头脑的时候.头脑就好好的休息
